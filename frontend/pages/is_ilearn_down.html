:D About Server Latency

Our background worker automatically tests the iLearn connection every 60 seconds. To prevent false alarms from network jitter, it sends 5 consecutive pings and averages the successful results. A red indicator appears at 0 ms only when all connection attempts fail completely. The Refresh button has a 15-second cooldown to avoid putting excessive load on iLearn servers.
