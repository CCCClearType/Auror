# 專案總覽

## 一、主題
跨平台數位遊戲發行與社群整合平台 (類 Steam 遊戲平台)。
平台名稱: Vapor&Auror (暫定)。

## 二、目的
打造一個便利且高擴充性的跨平台數位遊戲商城與社群中心。本系統對遊戲開發者與玩家，提供遊戲平台數位服務。玩家能在此進行遊戲瀏覽、消費購買、評論交流與成就解鎖。開發團隊能上架作品並獲得銷售數據。系統則透過資料庫統一管理玩家的虛擬資產 (遊戲庫)，以及跨裝置的無障礙偏好設定，確保各端操作體驗的一致性。

本系統旨在建立一個跨平台數位遊戲發行平台。為確保後續資料庫 CRUD 操作與 RESTful API 設計的明確性，本專題將系統目標具體定義為以下可測試的核心行為：
- **遊戲瀏覽**：玩家可搜尋遊戲列表，並查看特定遊戲的詳細資訊與定價。
- **購物車與願望清單**：玩家可將遊戲加入購物車進行結帳。
- **交易與授權生成**：玩家結帳購買後，系統需自動建立交易紀錄，並派發遊戲授權至玩家帳號。
- **遊戲庫管理**：玩家可在個人遊戲庫中查看已擁有授權的遊戲，或加入願望清單追蹤。
- **評論與檢舉機制**：玩家可對已購買的遊戲發表評價與評分。
- **產品上架管理**：開發者可上架新遊戲。
- **平台營運審核**：處理玩家的退款申請。
