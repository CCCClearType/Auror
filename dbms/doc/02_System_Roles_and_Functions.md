# 系統使用者與功能需求

## 三、系統使用者
本系統包含以下五種主要使用者角色：
1. **訪客 (Guest)**: 尚未擁有帳號或未登入的瀏覽者。
2. **登入的使用者 (User)**: 平台的使用者。
3. **開發者 (Developer)**: 負責遊戲專案開發、系統更新與內容維護的人。
4. **客服人員 (Customer Service Representative / CSR)**: 客服。
5. **管理員 (Admin)**: 負責平台管理與權限分派的最高管理者。

## 四、使用者需求功能

### 1. 訪客 (Guest)
- **基本功能**: 瀏覽首頁推薦、搜尋遊戲、查看遊戲詳情 (介紹、價格、公開評論)
- **帳號功能**: 登入、註冊

### 2. 登入的使用者 (User)
*(無限$$$$ 方便交易)*
- **基本功能**: 瀏覽首頁推薦、搜尋遊戲、查看遊戲詳情 (介紹、價格、公開評論)、登出
- **基本資料管理**: username, email, password
- **遊戲 & 商店模組**:
  - 買/退遊戲 (交易): 購買紀錄、將遊戲加入購物車、結帳購買、申請遊戲退款
- **社群與評論系統**: 對已擁有的遊戲發表評價
- **遊戲庫**: 個人遊戲庫、加入願望清單、移除願望清單
- **社交**: 發送/接受好友邀請、文字通訊、管理黑名單

### 3. 開發者 (Developer)
*(User + 上/下架遊戲 + 對自己的遊戲評價)*
- **基本功能**: 瀏覽首頁推薦、搜尋遊戲、查看遊戲詳情 (介紹、價格、公開評論)、登出
- **基本資料管理**: username, email, password
- **遊戲 & 商店模組**:
  - 管理遊戲: 上架遊戲、下架遊戲、上傳素材、看錢 (銷量)
  - 買/退遊戲: 購買紀錄、將遊戲加入購物車、結帳購買、申請遊戲退款
- **社群與評論系統**: 對已擁有的遊戲發表評價、對自己的遊戲發表評價
- **遊戲庫**: 個人遊戲庫、加入願望清單、移除願望清單
- **社交**: 發送/接受好友邀請、文字通訊、管理黑名單

### 4. 客服人員 (CSR)
*(User + 下架遊戲 + 對所有的遊戲評價 + 退錢)*
- **基本功能**: 瀏覽首頁推薦、搜尋遊戲、查看遊戲詳情 (介紹、價格、公開評論)、登出
- **基本資料管理**: username, email, password
- **遊戲 & 商店模組**:
  - 管理遊戲: 下架遊戲
  - 買/退遊戲: 購買紀錄、將遊戲加入購物車、結帳購買、申請遊戲退款
- **社群與評論系統**: 對所有的遊戲發表評價
- **遊戲庫**: 個人遊戲庫、加入願望清單、移除願望清單
- **社交**: 發送/接受好友邀請、文字通訊、管理黑名單
- **處理玩家退款申請**

### 5. 管理員 (Admin)
*(User + 下架遊戲 + 對所有的遊戲評價 + 管帳號)*
- **基本功能**: 瀏覽首頁推薦、搜尋遊戲、查看遊戲詳情 (介紹、價格、公開評論)、登出
- **基本資料管理**: username, email, password
- **帳號管理**: 
  - 停權帳號、移除帳號、管理帳號權限
  - 把 User / Developer 進化成 CSR / ADMIN
  - 把 CSR / ADMIN 退化成 User / Developer
- **遊戲 & 商店模組**:
  - 管理遊戲: 下架遊戲
  - 買/退遊戲: 購買紀錄、將遊戲加入購物車、結帳購買、申請遊戲退款
- **社群與評論系統**: 對已擁有的遊戲發表評價
- **遊戲庫**: 個人遊戲庫、加入願望清單、移除願望清單
- **社交**: 發送/接受好友邀請、文字通訊、管理黑名單
