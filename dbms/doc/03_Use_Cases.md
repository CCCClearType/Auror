# 系統使用案例 (Use Cases)

**角色縮寫與定義:**
- `ANY`: Guest, User, Developer, CSR, ADMIN
- `Logined`: User, Developer, CSR, ADMIN
- `Manager`: CSR, ADMIN

## 五、輸入資料、輸出資料與系統行為

| ID | 角色 (Actor) | 輸入 | 輸出 | 描述 / 行為 | 模組 |

| UC-01 | ANY | 移動到 home 網址 | 系統顯示符合條件的遊戲列表（包含名稱、封面縮圖與價格） | 瀏覽首頁推薦 | 基本權限與管理 |

| UC-03 | ANY | 移動到目標遊戲網址 | 系統顯示目標遊戲的簡介、價格、評論 | 查看遊戲詳情 (介紹、價格、公開評論) | 基本權限與管理 |
| UC-04 | Logined | 點擊「登出」按鈕 | 系統跳轉到 Guest 會顯示的平台頁面 | 登出 | 基本權限與管理 |
