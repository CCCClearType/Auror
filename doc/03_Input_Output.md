# 五. 輸入資料、輸出資料
ANY = Guest, User, Developer, CSR, ADMIN
Logined = User, Developer, CSR, ADMIN
Manager = CSR, ADMIN

| ID | Actor | 輸入 | 輸出 | 描述 / 行為 |
| --- | --- | --- | --- | --- |
| UC-01 | ANY | 移動到 home 網址 | 系統顯示符合條件的遊戲列表（包含名稱、封面縮圖與價格） | 瀏覽首頁推薦 |
| UC-02 | ANY | 搜尋關鍵字、標籤篩選條件 | 系統顯示符合條件的遊戲列表（包含名稱、封面縮圖與價格） | 搜尋遊戲 |
| UC-03 | ANY | 移動到目標遊戲網址 | 系統顯示目標遊戲的簡介、價格、評論 | 查看遊戲詳情 (介紹、價格、公開評論) |
| UC-04 | Logined | 點擊「登出」按鈕 | 系統跳轉到 Guest會顯示的平台頁面 | 登出 |
| UC-05 | Guest | 點擊「登入」按鈕 | 系統跳轉到登入頁面 | 登入 |
| UC-06 | Guest | 點擊「註冊」按鈕 | 系統跳轉到註冊頁面 | 註冊 |
| UC-07 | Logined | 新 Username | 系統顯示舊的 Username 替換成新的 | 基本資料管理 username |
| UC-08 | Logined | 新 email | 系統顯示舊的 email 替換成新的 | 基本資料管理 email |
| UC-09 | Logined | 新 password | 系統顯示舊的 password 替換成新的 | 基本資料管理 password |
| UC-10 | ADMIN | 選擇目標 User_ID | 系統顯示帳號已停權 | 停權帳號 |
| UC-11 | ADMIN | 選擇目標 User_ID | 系統顯示帳號已移除 | 移除帳號 |
| UC-12 | ADMIN | 選擇目標 User_ID | 系統顯示目標權限提升至對應權限 | 管理帳號權限 (提升) |
| UC-13 | ADMIN | 選擇目標 User_ID | 系統顯示目標權限降級至對應權限 | 管理帳號權限 (降級) |
| UC-14 | Logined | 點擊「遊玩」按鈕 | 系統開啟目標遊戲於使用者畫面 | 玩遊戲 |
| UC-15 | Logined | 點擊「下載」按鈕 | 系統顯示開始下載遊戲 | 載遊戲 |
| UC-16 | Logined | 移動到購買紀錄網址 | 系統顯示該使用者的購買紀錄 | 購買紀錄 |
| UC-17 | Logined | 點擊「放入購物車」按鈕 | 系統跳轉到購物車且遊戲已加入購物車 | 將遊戲加入購物車 |
| UC-18 | Logined | 點擊「購買」按鈕 | 系統跳轉到結帳頁面 | 結帳購買 |
| UC-19 | Logined | 選擇目標 Game_id | 系統顯示是否成功退款 | 申請遊戲退款 |
| UC-20 | Logined | 選擇目標 Game_id、評論、評價 | 顯示成功上傳評價 | 對遊戲發表評價 |
| UC-21 | Logined | 移動到遊戲庫網址 | 系統跳轉到個人遊戲庫 | 顯示個人遊戲庫 |
| UC-22 | Logined | 選擇目標 Game_id | 系統顯示遊戲已加入願望清單 | 加入願望清單 |
| UC-23 | Logined | 選擇目標 Game_id | 系統顯示遊戲已移除願望清單 | 移除願望清單 |
| UC-24 | Developer| 遊戲檔案、簡介、售價 | 系統顯示成功上架遊戲 | 上架遊戲 |
| UC-25 | Developer| Game_id | 系統顯示成功下架遊戲 | 下架遊戲 |
| UC-26 | Developer| Game_id、素材 | 系統顯示成功上傳素材 | 上傳素材 |
| UC-27 | Developer| 點擊「顯示數據」按鈕 | 系統顯示該遊戲面板 | 看錢銷量 |
| UC-28 | CSR | 點擊「同意」按鈕 | 系統顯示同意玩家退款 | 同意玩家退款申請 |
| UC-29 | CSR | 點擊「不同意」按鈕 | 系統顯示不同意玩家退款 | 不同意玩家退款申請 |
| UC-30 | Logined | 選擇目標 User_id | 系統顯示成功發送好友邀請 | 發送好友邀請 |
| UC-31 | Logined | 選擇目標 User_id | 系統顯示成功收回好友邀請 | 收回好友邀請 |
| UC-32 | Logined | 點擊「接受」按鈕 | 系統顯示成功接受好友邀請 | 接受好友邀請 |
| UC-33 | Logined | 點擊「拒絕」按鈕 | 系統顯示成功拒絕好友邀請 | 拒絕好友邀請 |
| UC-34 | Logined | 選擇目標 User_id | 系統顯示成功加入黑名單 | 加入黑名單 |
| UC-35 | Logined | 選擇目標 User_id | 系統顯示成功移除黑名單 | 移除黑名單 |
| UC-36 | Logined | 選擇目標 User_id, 訊息內容 | 系統顯示成功發送訊息 | 傳輸文字通訊給對方 |
| UC-37 | Logined | 點擊該使用者的通訊欄 | 系統顯示收到的訊息 | 顯示接收到的訊息 |
